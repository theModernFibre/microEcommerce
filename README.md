This is a microservice java spirngboot project integrated with react js and tailwind for user interface front-end.
This software artefact contains 7 services which are:
- discovery server: eureka
- api gateway: spring cloud gateway
- auth service: jwt authentication
- product service: products of eccomerce company/brand, crud and search
- order service: orders page for customers
- inventory service: inventory of products and crud functionality
- frontend : react and tailwind

  Deployment:
  Local: Docker with Kubernetes
  Non-Local: Render, using render to run multiple services
